A comprehensive and dynamic digital platform built to **preserve and promote India's rich cultural diversity** using modern technology.

## ✨ Core Innovations

This project stands out with its smart, proactive features:

| Feature | Description |
| :--- | :--- |
| **🤖 AI Chatbot ('Chatur')** | An interactive, conversational AI (BotPress) for real-time answers about Indian culture, history, and traditions. |
| **⚠️ Heritage Risk Radar** | A proactive tool that visualizes **real-time threats** (environmental, urban, etc.) to cultural heritage sites across India. |
| **🗺️ State-Wise Exploration** | Meticulously organized data covering heritage, traditions, and festivals for every Indian State and UT. |

## 💻 Tech Stack

The application is built on the robust **MERN Stack** for high performance and scalability.

| Component | Technology | Role |
| :--- | :--- | :--- |
| **Frontend** | **React.js, Tailwind CSS, Framer Motion** | Fast, responsive, and visually appealing user experience. |
| **Backend** | **Node.js, Express.js** | Handles API logic, user authentication, and feature integration. |
| **Database** | **MongoDB Atlas** | Flexible, cloud-hosted storage for diverse cultural data. |
